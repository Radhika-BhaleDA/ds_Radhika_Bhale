# Trader Behavior vs Market Sentiment Analysis

This project analyzes how cryptocurrency trader behavior aligns with or diverges from emotional market states such as Fear, Neutral, and Greed. Using real trade execution data combined with sentiment labels, this study explores whether traders make rational decisions or are influenced by psychological bias.
